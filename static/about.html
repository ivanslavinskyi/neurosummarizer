About the Project

NeuroSummarizer is an open-source tool that keeps neuroscientists up-to-date with the latest research. It automatically:

  • Fetches new papers from PubMed & arXiv
  • Generates concise AI-powered summaries using GPT
  • Categorises articles by research topic
  • Presents a clean, searchable, mobile-friendly feed

The code is evolving rapidly — feedback and pull-requests are welcome!
